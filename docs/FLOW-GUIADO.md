# Flujo guiado de uso

## 1. Crear proyecto

Un usuario con permiso de creación ingresa a Proyectos y presiona Nuevo. Completa datos principales y revisa la sección Responsables por etapa. Puede dejar los responsables automáticos o cambiarlos antes de guardar.

## 2. Asignación automática

Al guardar, cada etapa queda asignada a un usuario. La primera etapa queda activa y las demás quedan programadas.

## 3. Mis tareas

Cada usuario ingresa con su cuenta y ve:

- tareas activas;
- tareas programadas;
- aprobaciones pendientes;
- documentos observados;
- tareas de marketing.

## 4. Trabajo por etapa

Dentro de cada etapa el responsable puede:

- completar formulario;
- cargar documentos;
- marcar checklist;
- cambiar responsable si corresponde;
- solicitar aprobación;
- completar y avanzar.

## 5. Aprobaciones

Si la transición requiere aprobación, el flujo se detiene hasta que el aprobador resuelva. Si aprueba, se habilita la etapa siguiente. Si rechaza, la etapa queda en ajuste.

## 6. Cierre

El proyecto se cierra cuando el producto fue lanzado, marketing completó sus entregables y se registra el cierre.
